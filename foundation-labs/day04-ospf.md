# Day 4 — OSPF

## Learn
- OSPF is a link-state routing protocol
- Areas reduce routing overhead and organize topology
- DR and BDR are elected on multiaccess networks
- Neighbors form adjacencies before exchanging routes
- LSAs describe network topology

## Commands to Practice
- `show ip ospf neighbor`
- `show ip route ospf`
- `show ip ospf interface`

## Lab Goal
Build a 3-router topology and run OSPF in Area 0.

## Checklist
- [ ] Configure OSPF on all routers
- [ ] Verify neighbor relationships
- [ ] Confirm OSPF routes appear in the routing table
- [ ] Test reachability across the network
