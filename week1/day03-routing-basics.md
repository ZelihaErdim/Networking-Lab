# Day 3 — Routing Basics

## Learn
- Static routes are manually configured
- Default routes send unknown traffic to a next hop
- Routing tables show how the router decides where to send packets

## Commands to Practice
- `show ip route`
- `ping`
- `traceroute`

## Lab Tasks
- Configure static routes between two routers
- Add a default route
- Verify connectivity from one end device to another

## Challenge
Can you explain why `show ip route` shows multiple routes for the same destination?
