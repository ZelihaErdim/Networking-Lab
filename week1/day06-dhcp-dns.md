# Day 6 — DHCP + DNS

## Learn
- DHCP provides IP configuration automatically
- The DHCP exchange includes Discover, Offer, Request, and Ack
- DNS translates names to IPs
- Common DNS record types:
  - A record
  - CNAME
  - MX
  - NS

## Commands to Practice
- `nslookup google.com`
- `dig google.com`

## Lab Tasks
- Configure a DHCP server
- Verify clients receive addresses
- Configure DNS and test name resolution

## Notes
Write down what each DHCP message does and why DNS matters for troubleshooting.
