# Day 7 — Review Lab

## Topology
PC -> SW1 -> R1 -> R2 -> Server

## Requirements
- VLANs on the switch
- OSPF between routers
- DHCP for the PC
- DNS for name resolution

## Goal
By the end of the day, you should be able to:
- explain why the PC gets an IP
- explain how the routers reach each other
- explain how the server is reachable
- troubleshoot if one part fails
