# Week 1 — Core Networking Foundations

This section highlights the networking concepts most important for understanding how traffic moves and how problems are diagnosed.

## Focus areas
- switching and VLAN design
- STP and redundancy control
- routing basics and reachability
- OSPF and route learning
- BGP awareness and peer relationships
- DHCP and DNS fundamentals

## Lab topics
- VLANs and switching
- STP loop prevention
- basic routing
- OSPF configuration
- BGP basics
- DHCP and DNS practice
- end-to-end review lab

## Core commands practiced
- `show vlan brief`
- `show mac address-table`
- `show interfaces trunk`
- `show spanning-tree`
- `show ip route`
- `show ip ospf neighbor`
- `show ip bgp summary`

## Why this matters for a portfolio
These exercises show that I understand not only the syntax of commands, but also how networks behave under normal and failure conditions.
