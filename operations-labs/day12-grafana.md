# Day 12 — Grafana Monitoring

## Learn
- Dashboards help visualize system and network health
- Alerts notify you when metrics cross thresholds
- Useful metrics include:
  - CPU
  - memory
  - interface utilization
  - ping latency

## Goal
Build a dashboard that shows:
- CPU usage
- memory usage
- interface traffic
- ping latency

## Practice Tasks
- Install Grafana
- Add a data source
- Create at least one dashboard panel for network health
